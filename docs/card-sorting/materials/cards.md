# Printable Cards – Open Card Sort

Print single-sided on card stock and cut into individual cards. Include 10–12 blank cards for participants to add items you missed. Keep only the content labels below; do not provide predefined categories (open sort).

Recommended set (40–50 cards):

1. Getting started (new to CS site)
2. What class should I take next?
3. Course list (A–Z)
4. Prerequisite chains
5. Class schedules and sections
6. Registration how-to
7. Add/drop deadlines
8. Waitlists
9. Prerequisite override/permission-to-add
10. Degree requirements – BS in CS
11. Degree planner / 4-year plans
12. Graduation checklist
13. Minor in CS
14. Transfer credit evaluation
15. AP/IB credit
16. Course syllabi (archive)
17. Textbooks and materials
18. Labs: locations and hours
19. Computing resources (lab machines, VPN)
20. Software licenses (e.g., IDEs, tools)
21. TA hours
22. Tutoring center
23. Office hours directory
24. Student FAQs
25. Internship credit (e.g., CS 399R)
26. Capstone project (e.g., CS 480)
27. Events calendar
28. Department news
29. Jobs and internships board
30. Staff/faculty directory
31. Code of conduct
32. Contact us
